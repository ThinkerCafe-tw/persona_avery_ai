# **⏺ 露米 3.0 產品介紹資料**

## ** 什麼是露米？**

露米是你的**專屬 AI 心靈夥伴**。

她不只是聊天機器人，而是一位**懂你情緒、陪伴成長的智慧夥伴**。

透過 LINE 即可使用，24 小時隨時陪你聊天、療癒、思考、反省與快樂！

## ** 如何開始使用？**

1. **加入LINE好友**：掃描 QR Code 或搜尋 ID
2. **開始對話**：隨意說你想說的話
3. **體驗不同人格**：露米會自動切換最適合的回應風格
4. **生成日記**：只要說「總結今天的日記」，就能獲得一篇完整的心情筆記！
